🍽️ Menu Paket Bergizi Balita (0–6 Tahun)

Menu disusun dengan tekstur lembut, gizi seimbang, dan porsi sesuai kebutuhan balita dengan harga gratis. Maksimal 3 porsi/orang/hari

⚠️ Kuota Hari Ini: 3000 porsi tersedia
Pilih maksimal 3 menu favorit (batas per orang/hari), lalu klik tombol untuk lanjut!

🍗 Pilih Menu Bergizi (600-700 kcal/porsi)

🍲

Nasi Tim Ayam Sayur

Nasi tim (40g) + Ayam cincang (30g) + Wortel (20g) + Bayam (15g) + Tahu (25g) + Pisang (50g) + Susu (150ml)

Total: ~630 kcal
Vit A, C, Kalsium, Protein Tinggi
🥔

Bubur Kentang Telur

Kentang (80g) + Telur ayam (50g) + Wortel (20g) + Tempe (25g) + Pepaya (50g) + Susu (150ml)

Total: ~620 kcal
Kalium, Probiotik, Energi Cepat
🐟

Tim Ikan dan Sayur

Nasi tim (40g) + Ikan (30g) + Labu kuning (25g) + Brokoli (20g) + Tahu/tempe (25g) + Jeruk (50g) + Susu (150ml)

Total: ~680 kcal
Zat Besi, Vit C, Anti Anemia
🍲

Sup Ayam Lembut

Nasi lembek (40g) + Ayam suwir (30g) + Wortel (20g) + Kentang (40g) + Jagung (20g) + Semangka (50g) + Susu (150ml)

Total: ~620 kcal
Zat Besi, Vit A & C, Kalsium
